Instructions

To create a new transcript application :-

  1. Click on "New Application" above to go to the application form
  2. Fill in the details and submit the form
  3. Make transaction when prompted to do so

To access submitted transcript applications :-

  1. Click on "View Transcript Applications" above
  2. Select desired application from the list of applications
  3. If application is accepted, downloadable pdf file containing the transcript should appear
